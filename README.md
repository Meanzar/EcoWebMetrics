# EcoWebMetrics

# Dependencies

- [ESLint](#lint)
- [Jest](#test)

## Lint

[ESLint](https://eslint.org/docs/latest/use/getting-started)

Configure the rules in the **.eslintrc.json** file

Run `npm run lint`

## Test

[Jest](https://jestjs.io/docs/getting-started)

You can change the test in the `__test__` folder

Run `npm run test`
